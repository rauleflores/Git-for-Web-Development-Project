    1. What is Semantic HTML? 
        HTML tags that have actual meaning. They are used to clearly define what each part of the webpage is.
    
    2. What is HTML used for? 
        HTML is used to build the raw structure of a webpage.
    
    3. What is an attribute and where do we put it? 
        An attribute gives extra information about an element such as height and width. It is placd into the opening tag of an element.
    
    4. What is the h1 tag used for? How many times should I use it on a page?
        The H1 tag is used to signify that the tagged text is the most important on the page. It should be used only once, preferably at the top of the page.
    
    5. Name two tags that have required attributes
        <input> and <textarea> both require attributes.

    6. What do we put in the head of our HTML document? 
        Meta data such as the title, author information, styles, scripts and character sets.

    7. What is an id?

    8. What elements can I add an id to?

    9. How many times can I use the same id on a page?

    10. What is a class?

Stretch Questions

    1. What is the difference between an inline element and a block element?
        A block element starts on a new line by default and takes up the available width.
        An inline element only uses as much space as it needs and doesn't start a new line.
 
    2. What happens when an element is positioned absolutely? 
        The element is able to be made to behave differently than normal because it is no longer apart of the normal document layout flow.

    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
        Use display:inline-block;
